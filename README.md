# WhatsApp Auto Reply (Gemini)

A lightweight system that automatically replies to WhatsApp messages using Google Gemini to generate intelligent, context-aware responses.

## Overview
This project listens for incoming WhatsApp messages, processes their content, sends relevant context to the Gemini API, and replies automatically with AI-generated responses. It is designed for experimentation, productivity, and learning purposes.

## Features
- Automatic WhatsApp message replies
- AI-generated responses using Gemini
- Context-aware conversation handling
- Configurable reply rules and limits
- Simple and extensible design

## How It Works
1. A WhatsApp client detects incoming messages
2. Message text is processed and filtered
3. Content is sent to Gemini as a prompt
4. Gemini generates a response
5. The response is sent back automatically

## Tech Stack
- Node.js
- Google Gemini API
- WhatsApp automation client
- JavaScript / TypeScript

## Usage
- Configure your Gemini API key
- Run the system
- Incoming WhatsApp messages are replied to automatically
